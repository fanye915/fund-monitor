# 三地基金实时监控系统

一个基于GitHub Pages的实时基金监控网站，展示A股、港股、美股三个基金组合的实时数据。

## 功能特点
- 实时显示三个基金组合的持仓情况
- 自动计算成本价、现价、收益率等关键指标
- 30秒自动刷新数据
- 响应式设计，支持手机和电脑访问
- 净值曲线图表展示

## 数据来源
使用 [iTick API](https://itick.org) 获取实时行情数据：
- A股数据：上海/深圳交易所
- 港股数据：香港交易所
- 美股数据：纽约/纳斯达克交易所

## 快速开始

### 1. 注册iTick API
1. 访问 https://itick.org 注册账号
2. 获取免费的API Token
3. 在 `api/config.js` 中替换 `your_itick_token_here`

### 2. 部署到GitHub Pages
1. 在GitHub创建新仓库，命名为 `fund-monitor`
2. 将所有文件上传到仓库
3. 进入仓库设置 → Pages
4. 选择 `main` 分支作为源
5. 保存后等待几分钟，网站即可访问

### 3. 本地运行
1. 克隆仓库到本地
2. 用浏览器打开 `index.html`
3. 或使用本地服务器：`python -m http.server 8000`

## 基金配置
系统包含三个基金组合：
- **A股基金**：100万人民币，5只ETF
- **港股基金**：100万港币，5只ETF
- **美股基金**：100万美元，5只ETF

## 技术栈
- HTML5 / CSS3 / JavaScript (ES6+)
- Chart.js 数据可视化
- iTick API 实时数据
- GitHub Pages 免费托管

## 注意事项
1. API Token 请妥善保管，不要公开分享
2. 免费版API有调用频率限制（10次/秒）
3. 股市交易时间外数据可能不更新
4. 汇率数据为简化版本，实际应用中应从API获取
